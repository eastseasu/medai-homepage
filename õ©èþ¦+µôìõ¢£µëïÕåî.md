# MedAI 主页 — 上线操作手册

> **域名：** starpuremed.com（腾讯云注册中，审核预计1-3工作日）
> **方案：** 境外 Vercel 快速上线 → 备案通过后切换至腾讯云 COS
> **适用文件：** index.html / privacy-policy.html / terms.html

---

## 阶段一：Vercel 快速上线（域名注册后立即可做）

### 前置条件
- 域名已购买（腾讯云/阿里云/其他均可）
- 有 GitHub 账号（用于托管代码）

### 步骤

#### 1. 上传代码到 GitHub

1. 打开 [github.com](https://github.com) 新建仓库，命名如 `medai-homepage`，选 **Public**
2. 把 `medai-homepage/` 目录下所有文件上传到仓库根目录
   - index.html
   - privacy-policy.html
   - terms.html
   - vercel.json

#### 2. 部署到 Vercel

1. 打开 [vercel.com](https://vercel.com)，用 GitHub 账号登录
2. 点击 **Add New → Project**
3. 选择刚才创建的 `medai-homepage` 仓库
4. Framework Preset 选 **Other**
5. 点击 **Deploy** → 等待约 1 分钟部署完成
6. 得到一个类似 `medai-homepage-xxx.vercel.app` 的地址，可直接访问 ✅

#### 3. 绑定自定义域名（Vercel 侧）

1. 进入 Vercel 项目 → **Settings → Domains**
2. 输入 `starpuremed.com` 和 `www.starpuremed.com`
3. Vercel 会提示需要添加的 DNS 记录（CNAME 或 A 记录）

#### 4. 配置 DNS 解析（域名注册商侧）

根据 Vercel 提示，在域名注册商控制台添加：

| 记录类型 | 主机记录 | 记录值 |
|----------|---------|--------|
| CNAME | www | `cname.vercel-dns.com` |
| A | @ | `76.76.21.21` |

> DNS 生效时间：一般 5-30 分钟，最长 48 小时

**此时网站已可通过自定义域名全球访问，Vercel 自动提供免费 HTTPS 证书 ✅**

## 阶段二：腾讯云 COS 正式上线（备案通过后）

### 步骤

#### 1. 创建 COS 存储桶

1. 登录 [腾讯云控制台](https://console.cloud.tencent.com/cos)
2. 创建存储桶：
   - 名称：`medai-homepage`
   - 地域：**北京**（就近原则）
   - 访问权限：**公有读私有写**
3. 进入存储桶 → **基础配置 → 静态网站**
4. 开启静态网站，索引文档填 `index.html`，错误文档填 `index.html`

#### 2. 上传文件

将以下文件上传至存储桶根目录：
- index.html
- privacy-policy.html
- terms.html

#### 3. 开通 CDN + 绑定域名

1. 进入 **[内容分发网络 CDN](https://console.cloud.tencent.com/cdn)**
2. 添加域名，源站选择 COS 存储桶地址
3. 开启 **HTTPS**（申请免费 SSL 证书，腾讯云一键申请）

#### 4. 修改 DNS 解析（切换时机）

备案通过后，在域名控制台将 DNS 记录从 Vercel 切换至腾讯云 CDN：

| 记录类型 | 主机记录 | 记录值 |
|----------|---------|--------|
| CNAME | www | 腾讯云 CDN 提供的 CNAME 地址 |
| CNAME | @ | 腾讯云 CDN 提供的 CNAME 地址 |

> **切换过程对用户无感知，DNS TTL 设为 600 秒可加快切换速度**
